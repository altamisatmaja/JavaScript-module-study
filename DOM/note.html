DOM: Document Object Models

Interface pemrograman untuk dokumen web.
DOM mewakili halaman sehingga program dapat mengubah struktur, gaya, dan konten dokumen. Selain itu, DOM juga mewakili dokumen sebagai node dan objek.

DOM menetapkan metode querySelectorAll dalam kode ini harus mengembalikan daftar semua elemen

HTML DOM

Model objek standar dan interface pemrograman untuk HTML DOM mendefinisikan beberapa hal, antara lain: 1. Elemen HTML sebagai objek 2. Properti dari semua elemen HTML 3. Metode untuk mengakses semua elemen HTML 4. Events untuk semua elemen HTML In other words, DOM HTML is standard for get, update, create, and delete;
